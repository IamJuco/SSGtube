## [2024] 안드로이드 유튜브 앱 제작  
유튜브 API를 활용한 동영상 리스트 어플입니다.<br>
사용자에게 YouTube의 현재 인기 및 새로운 콘텐츠를 중점적으로 보여주고, 콘텐츠를 검색하고, 보관함에서 저장 및 삭제를 할 수 있습니다.

## 개요
- 프로젝트 이름: `쓱튜브 (SSG Tube)`
- 프로젝트 기간: `2024-05-13 ~ 2024-05-23`
- 프로젝트 핵심기술
    - `MVVM`
    - `ViewModel,Live Data`
    - `Repository Pattern`
    - `Retrofit`
